### 🚀 Projeto em destaque

#### [Cloud Move Analyzer](https://cloudmoveanalyzer.com/)
Ferramenta gratuita que checa se recursos do Azure podem ser movidos entre **Resource Groups, Subscriptions e Regions** — um problema clássico em projetos de migração e reorganização.

- 📊 Base de **747+ tipos de recurso** atualizada **semanalmente** a partir da documentação oficial da Microsoft
- 🌍 Disponível em **8 idiomas** (EN, PT-BR, ES, FR, ZH-CN, AR, RU, JA)
- ⚡ 100% client-side (vanilla JS), hospedado no **Azure Static Web Apps**
- ✅ CI completo: testes Vitest + PowerShell, lint HTML, deploy automático
- 📦 [Repositório](https://github.com/jrlimax/CloudMoveAnalyzer)
